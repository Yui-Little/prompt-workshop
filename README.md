📱 主要功能

· ✨ 生成提示词：输入你的粗略想法，AI 自动补全为结构完整、可直接使用的提示词。
· 🔄 优化提示词：粘贴已有的提示词，AI 会增强清晰度、约束条件和可执行性。
· 🧠 智能策略：无需手动选择“写作/编程/设计”等类别，AI 先理解意图再自适应生成。
· 📋 最近记录：自动保存最近 12 条生成/优化记录，方便回溯。
· ⚙️ 接口设置：支持自定义 API Key、Base URL、模型和 Temperature，兼容 OpenAI 及第三方接口。

🚀 快速开始

1. 配置 API

首次打开会自动弹出设置面板，你需要填写：

· API Key：你的 OpenAI 或兼容接口的密钥（仅保存在浏览器本地）。
· Base URL：接口地址，需填写到 /v1 级别（例如 https://api.openai.com/v1）。
· Model：模型名称（如 gpt-4o-mini、claude-3-haiku 等）。
· Temperature：控制随机性（0~1），建议保持 0.7。

填写完成后点击 保存设置，即可开始使用。

2. 生成提示词

在左侧（手机端为上方）输入框中描述你的需求，例如：

```
我想做一个“每周输出行业洞察”的工作流，请帮我写一个通用提示词：
- 我会提供：行业主题、目标读者、可用数据来源
- 我希望输出：观点摘要、关键证据、可执行建议、风险提示
- 结果风格：专业但易读，适合公众号和内部汇报
- 约束：要有结构化格式，可重复使用
```

点击 生成 按钮，AI 会在数秒内返回一个完整的提示词。

3. 优化提示词

将已有的提示词粘贴到右侧（手机端为下方）输入框，点击 优化 即可获得改进版本。

4. 快捷操作

· 复制结果：点击结果区域旁的复制按钮。
· 送去优化：将生成的结果直接填入优化输入框。
· 清空：清空当前输入和输出。
· 快捷键：在输入框中按 Ctrl+Enter 或 Cmd+Enter 可快速提交。

❓ 常见问题

Q：为什么我点击生成后没反应？
A：请检查 API 设置是否正确（Base URL 是否以 /v1 结尾，API Key 是否有效），并打开浏览器控制台查看是否有跨域错误。部分接口可能不支持浏览器直接调用，需确认服务端已配置 CORS。

Q：支持哪些模型？
A：只要接口兼容 OpenAI 的 /v1/chat/completions 格式，均可使用。例如 OpenAI 全系列、Anthropic 通过第三方代理、本地部署的 ollama 等。

Q：历史记录可以保存多久？
A：最近 12 条记录会保存在浏览器中，手动清空或清除浏览器数据后会丢失。
